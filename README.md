# LLM-Diagnostic-Accuracy-Analysis-on-the-Trustworthy-AI-Psychotherapy-Dataset
I reproduced and extended the diagnostic-accuracy analysis from a CIKM '25 paper on multi-agent LLM workflows for mental health diagnosis, comparing how Llama-4, Mistral-Saba, Qwen-QWQ, and GPT-4.1-Nano perform on ~8,000 simulated DSM-5 diagnostic interviews across 10 disorder categories.
